# cineplus
Proyecto que crea el desarrollo de un cine, creada en python con wxglade, con 4 ventanas: incio de sesion,  taquilla, dulceria y ticket, para que el usuario pueda acceder.

Ventana inicio de sesion
--El sistema debe de tener un inicio de sesion.
--Para este proyecto, el inicio de sesion se hara con los datos preestablecidos (por ejemplo, usuario:cajero y contraseña 123) no es necesario implementar una base de datos.

Venta de boletos (Taquilla)
--el sistema debe permitir al empleado seleccionar la pelu¿icula, el horario y la sala.
"Anabelle" sala 1 
"El telefono negro" sala 2
"rio" sala 3 
se debe especificar la cantidad de boletos
Se debe indicar si los boletos son para adultos o para menor, ya que el precio es diferente(adutos $50, menores $35)
--Al finalizar la seleccion de boletos, el sistema debe de mostrar un resumen de los boletos adquiridos y calcular el subtotal de la taquilla.

Venta de productos (Dulceria)
--el sistema debe permitir al empleado seleccionar el combo, el tamaño y la cantidad
"combo indivual" 
"combo familiar"
"combo jumbo"
"palomitas chicas medianas y grandes"
"refresco, el sabor, la cantidad y el tamaño"
Se debe indicar cuantos combos son para poder realizar el total de toda la compra.
--Al finalizar la seleccion de combos, el sistema debe de mostrar un resumen de los combos o productos adquiridos y calcular el subtotal de la dulceria.

Ventana ticket
El sistema debe de permitir la opcion de sumar y hacer un resumen de todas las compras anteriores..
--Al finalizar la seleccion de boletos, dulceria, el sistema debe de mostrar un resumen de los boletos adquiridos, de productos adquiridos y calcular el subtotal de la compra de todo.
